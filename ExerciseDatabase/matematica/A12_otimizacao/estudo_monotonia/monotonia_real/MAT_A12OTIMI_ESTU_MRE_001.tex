\exercicio{
Uma pequena empresa vende $x$ unidades de um produto por dia. O lucro diário (em euros) é dado pela função:
\[
L(x) = -2x^2 + 80x - 200
\]
onde $x$ representa o número de unidades vendidas.
}

\subexercicio{Calcula a derivada $L'(x)$ que representa a taxa de variação do lucro.}

\subexercicio{Determina para que valores de $x$ o lucro está a aumentar (função crescente).}

\subexercicio{Determina para que valores de $x$ o lucro está a diminuir (função decrescente).}

\subexercicio{Quantas unidades deve a empresa vender para maximizar o lucro? Qual é esse lucro máximo?}
